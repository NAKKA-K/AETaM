# AETaM(いーたん)
毎日の食事、会話によって、性格や体型、体調が変化する成長型AIです  

# Description
毎日の食事、会話によって、性格や体型、体調が変化する成長型AIです。   
AETaMへの食事は、自分の食事画像をあげましょう。  
画像を元にAETaMの健康状態は日々変換していきます。  
AETaMとも会話は、AETaMの性格を育てることになります。  
自分の好みの性格に育てたり、いつも通り会話してどんな性格に育つのか観察したりして見ましょう。
